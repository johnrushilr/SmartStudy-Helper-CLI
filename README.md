**SmartStudy Helper CLI**



*This is an AI-powered CLI application that helps students to manage and access notes, create/generate summaries, and create quizzes to test their knowledge*.



\- Features of this code

&#x20; 1. Add and view notes

&#x20; 2. AI-based summarization

&#x20; 3. Quiz generation

&#x20; 4. CLI-based interaction



\- Requirements

&#x20; 1. Desktop/laptop

&#x20; 2. Python Interpreter



\- How to run the program



&#x20; 1. Copy repository files from GitHub

&#x20; 2. Navigate to folder

&#x20; 3. Run main.py
